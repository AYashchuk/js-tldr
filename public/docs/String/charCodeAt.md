# String.prototype.charCodeAt()

The `charCodeAt` method returns an _integer_ between **0** and **65535** representing the UTF-16 code unit at the given _index_ of a _string_.

## Syntax

```js
const integer = string.charCodeAt(index);
```

## Usage examples

```js
'ABC'.charCodeAt(0); // -> 65
```
